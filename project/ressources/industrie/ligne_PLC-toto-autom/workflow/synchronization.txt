SYNCHRONISATION GLOBALE
Signals and handshakes:
- M1.ready_for_pick -> M2.request_pick
- M2.pick_done -> M1.cycle_release
- M2.load_done -> M3.start_glue
- M3.glue_done -> M4.start_screw
- M4.screw_ok -> M5.trigger_vision
- M5.vision_ok -> M6.start_mark
- M6.mark_ok -> M7.pick
- M7.pallet_done -> PLC.cycle_finished

Timing constraints per handshake (nominal):
- M1->M2: <= 150 ms
- M2->M3: <= 200 ms
- M4->M5: <= 120 ms

Profinet segments: A (M1,M2,M3), B (M4,M5,M6,M7)
